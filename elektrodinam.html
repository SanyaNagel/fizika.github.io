"Электродинамика (Электричество и магнетизм)"

Введение:

Электростатика - раздел физики, изучающий электрические поля неподвижных зарядов (Заряженых тел).

1.Закон Кулона:

F=q1q2kr2

Где:
q1 - величина первого заряда
q2 - величина второго заряда
r - расстояние между зарядами

k = 1   4πε0 ε0 = 8,85×10-12

2.Напряженность электрического поля (E):

E=Fq

Где:
F - сила действующая на заряд
q - величина заряда

3.Электроёмкость (С) - показывает, какой по модулю заряд будет на каждой из обкладок конденсатора, если разность потенциалов между ними равна 1 В:

C=qU

q - заряд
U - разность потенциалов (φ1-φ2)

4.Напряженность поля в конденсаторе:

E = q   εε0S

Где: q - модуль заряда одной из пластин (Кл).
ε - диэлектрическая проницаемость диэлектрика
ε0 = 8,85×10-12 (Ф/м) - электрическая постоянная
S - Площадь одной из пластин (м2)

5.Электроёмкость плоского конденсатора:


C=εε0Sd

Где:
d - толщина диэлектрика.
ε - диэлектрическая проницаемость диэлектрика
ε0 = 8,85×10-12 (Ф/м) - электрическая постоянная
S - Площадь одной из пластин (м2)

6.Энергия заряженного конденсатора:

W=q22C=U2C2=qU2

Где:
U - разность потенциалов.

"Постоянный ток"

7.Сила тока (I):

I=qt

Где:
q - заряд
t - время

8.Электродвижущая сила (ЭДС):

ε=Aстq

Где:
A - работа по перемещению заряда совершённая сторонними силами
q - величина заряда

9.Закон Ома для участка цепи:

I=UR

10.Сопротивление проводника (R):

R=ρlS

ρ - удельное сопротивление вещества (Ом×м)
l - длинна проводника
S - площадь поперечного сечения проводника (м2).

11.Закон Ома для полной цепи, состоящей из источника тока с ЭДС и внутренним сопротивлением r и внешним сопротивлением R

I= εR+r


Где:
ε - эдс источника.
R - внешнее сопротивление источника
r - внутреннее сопротивление источника

12.Работа электрического тока на участке цепи за время t:

A=IUt

Где:
I - сила тока (Ампер).
U - напряжение t - время

13.Мощность электрического тока на участке цепи равна:

P=At=IU

Где:
A - работа электрического тока
t - время работы тока
I - сила тока
U - напряжение

14.Закон Джоуля - Ленса:

Q=I2Rt

Где:
I - Сила тока
R - внешнее сопротивление
t - время
